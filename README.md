Car Information Manager / Car Profile Viewer (Python)
    This project is a simple Car Information Manager built using Python and Object-Oriented Programming (OOP).
    It allows users to create a car profile, view complete details, update mileage, and calculate the age of the car.
Features:
   1. Car Profile Creation
      Each car object stores:
          - Manufacturer name
          - Model name
          - Manufacturing year
          - Current mileage

  2. Display Car Information
     - The program prints all car details in a clear and readable format.

  3. Update Mileage
     - Mileage updates only if the new mileage is higher.
     - Ensures realistic car mileage progression.

  4. Calculate Car Age
     - Computes the age of the car based on the current year provided by user.


Concepts Used
  This project helps beginners understand:
      - Python classes & objects
      - Constructor (__init__)
      - Class attributes
      - Methods and condition checks
      - Real-world OOP application
